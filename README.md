## 💡 The Problem
CoreInventory

## 🚀 Our Solution
InventoryPro is a streamlined, dark-themed web application designed to make inventory management fast, visual, and secure. It allows warehouse managers to track the complete lifecycle of a product from the moment it arrives at the loading dock to the moment it ships to a customer.

## ✨ Key Features
* **Real-Time Dashboard:** Get a bird's-eye view of late packages, ongoing operations, and pending deliveries.
* **Inbound & Outbound Tracking:** Dedicated modules for **Receipts** (vendors) and **Deliveries** (customers) with dynamic stage tracking (Ready, Draft, Waiting).
* **Internal Stock Transfers:** Seamlessly move partial or full stock between different warehouse locations with built-in validation.
* **Move History Ledger:** A complete, unalterable log of all stock movements (In, Out, Transferred) with visual indicators.
* **Secure Profile Management:** Features a simulated multi-factor authentication (MFA) flow requiring an OTP to reset user passwords.
* **Responsive Dark UI:** A custom-built, glassmorphism-inspired dark mode that reduces eye strain for warehouse workers on night shifts.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Grid/Flexbox UI), Vanilla JavaScript
* **Backend:** Python (Flask)
